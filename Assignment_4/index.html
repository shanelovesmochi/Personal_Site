Population Ageing

State - County - Census Tract

The proportion of the population aged 60 and above reaches 10% of the total population, which is the standard for a country or region to enter an aging society (or aging population). I am using the S0101 Age and Sex dataset from the U.S. Census Bureau to analyze the percentage of the population aged 60 and above in each state. By calculating and mapping this percentage, I aim to identify states where the population is aging more significantly. This analysis helps assess whether local communities are demographically dynamic or experiencing an aging trend. To classify the data, because the data on the aging population is relatively concentrated, I use Natural Breaks (Jenks), which optimally groups similar values while maximizing the differences between classes. This method ensures that patterns in aging populations are more visually distinguishable, making it easier for the eye to interpret regional variations effectively.
